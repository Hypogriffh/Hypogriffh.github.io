The following is a message from the entity known as Gino.


Hello hello, this is Gino coming straight from the airwaves to your lovely computer. How tidy and spacious! As of today, I figured I'd migrate my gamedev blog from GameJolt to here. I'll still be posting copies of the blog there of course, but I'll consider this site the proper "home" of the blog. So hello! Nice to meet you.

This blog is partially inspired in name by The Word of Notch, of course. Just a cozy place where I can talk about the games I'm working on without worrying about character limits and other restrictions. I could embed a whole game in a blog post if I wanted!

What frightening power to wield at my fingertips.

Will I do that? Nah, probably not. For now, I'll start by talking about my general plans for the future gamedev-wise.

Current Work

Currently, I'm easing back into working on a few games. I'm a member of a FNAF fan-game's dev team, but I've been distant since my hiatus in the second half of 2024 through last month. It's a whole lot of "I'll talk more about it soon", unfortunately. My projects are in a constant state of "not quite done enough to talk about, but I REALLY want to talk about them". Should have something more soon(tm).

What I can talk about is a very fun practice project I've been working on, called AMGL (Another MonoGame Library)! It's a project that aims to speed up the MonoGame workflow with a built-in data-driven Entity-Component System, a handy Mathf class derived from Unity, and more. It's currently in a decent state to make some games with, but I'm not quite satisfied enough for a public release yet. I'll keep track of development on this site, though!

In the same vein as AMGL, I'm also tinkering with a project currently called Caw, an RPG Maker-esque engine forked over from AMGL's code. It trims the things from AMGL that aren't necessary for a simple top-down RPG and adds a proper editor application and project system. Caw isn't nearly as close to completion as AMGL, since it already hinges on AMGL itself being completed. We'll see!

Future Work

I have a constant stream of some-good, most-bad game ideas running through my head every single day. I have the silly habit of picking up and dropping projects within a week or two of each other. So, these aren't set in stone. I might forget them-

  1. "Eternus" - A multiplayer top-down sandbox RPG set on a collection of islands. Terraria-likes run in my blood, I can't escape...
  2. "Gino Maker" - Make your own little TV head friend! I'll probably host the "game" on Itch and this website.

That's about all I have, I think. Hopefully I'll get better at writing blog posts in the future. And hopefully I remember to post at all! Anyway, that's it. Drink water! Don't forget it!


Message concluded.
